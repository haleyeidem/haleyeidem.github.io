# haleyeidem.github.io

Personal portfolio site for Haley Eidem — data storyteller, quantitative researcher, designer.

**Live (once deployed):** https://haleyeidem.github.io/

## Stack

One hand-written `index.html` — no framework, no build step. Fonts from Google Fonts; everything else is inline CSS/SVG/JS.

## Adding the Dill's Litter Log screenshots

Drop three phone screenshots into `assets/` named exactly:

- `assets/dills-01.png` — home tab (status card + scatter chart)
- `assets/dills-02.png` — analytics heatmap
- `assets/dills-03.png` — trends / anomaly flags

The site shows dashed placeholders until the files exist. Any aspect ratio works; they're cropped to a phone frame.

## Editing

Everything lives in `index.html`. Sections are marked with comments (`<!-- 01 · Odonata -->` etc.). Open the file in a browser to preview changes — no server needed.
